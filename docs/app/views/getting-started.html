ngGovUk Introduction

The ngGovUk module is designed to be a platform wide Bootstrap wrapper that applies the GOV.UK elements styling requirements to your Angular JS project.

Also included are a number of Angular JS modules that have common functionality that may be useful in your project.

Getting Started

Dependencies

This repository contains a set of native AngularJS directives based on Bootstrap's markup and CSS. As a result no dependency on jQuery or Bootstrap's JavaScript is required. The only required dependencies are:

  • AngularJS (requires AngularJS 1.2.x, tested with 1.2.16)
  • UI Bootstrap
  • Bootstrap CSS (tested with version 3.1.1). This version of the library (0.12.0) works only with Bootstrap CSS in version 3.x. 0.8.0 is the last version of this library that supports Bootstrap CSS in version 2.3.x.
